Actividad Unidad 1 - Tarea 1 (Prueba de aplicaciones)

En esta tarea se analiza, corrige, ejecuta y testea una aplicación desarrollada en Python

Índice

  • Objetivos
  • Desarrollo
  • Reflexión

Objetivos

Para la realización de esta tarea se tiene en cuenta el cumplimiento de los siguientes objetivos:

  • Analisis y comprensión del código

  • Uso de herramientas del IDE para ejecución, depuración paso a paso y seguimiento del flujo de control

  • Diseñar, implementar y ejecutar pruebas unitarias que cubran las funciones y lógica del programa

  • Ejecución del programa en un entorno controlado para simular su comportamiento en un contexto real

Desarrollo

La actividad se ha desarrollado en las siguientes fases:

  1. Documentación y prueba del programa

  2. Realización de los test unitarios del programa

  3. Ejecucion del programa en un entorno controlado (sandbox)
